# [순시미 맞고] UI/UX 디자인 가이드

## 1. 디자인 컨셉 (Design Philosophy)
- **키워드:** 네오 레트로(Neo-Retro), 다크 모드, 사이버펑크 한 스푼, 찰진 타격감(Juicy).
- **탈피 대상:** 촌스러운 초록색 담요 배경, 올드한 궁서체, 정적인 움직임.
- **핵심 목표:**
  1. **화투패 강조:** 어두운 배경을 사용하여 화투패의 원색(빨강, 노랑)이 네온 사인처럼 돋보이게 한다.
  2. **유리 질감 (Glassmorphism):** 팝업창이나 패널은 반투명한 유리 느낌을 주어 세련됨을 더한다.
  3. **반응성:** 버튼을 누르거나 패가 붙을 때, 화면이 살짝 튀거나 떨리는 애니메이션으로 '손맛'을 구현한다.

---

## 2. 컬러 팔레트 (Color Palette)
Flutter의 `Color(0xFF...)` 포맷에 맞춰 정확한 값을 사용한다.

### 2.1 배경색 (Backgrounds)
- **메인 배경 (App BG):** `#1A1E23` (아주 깊은 차콜색) ~ `#131519` (하단으로 갈수록 어두워지는 그라데이션)
- **게임 매트 (Floor):** `#233038` (채도가 낮은 다크 청록색 - 눈이 편안함)
- **유리 패널 (Glass):** `#FFFFFF` (투명도 10% + Blur 효과)

### 2.2 포인트 컬러 (Accents)
- **브랜드/액션 (GO):** `#FF4B4B` (형광빛이 도는 쨍한 레드오렌지) - '고', '치기' 버튼
- **보조/방어 (STOP):** `#4B7BFF` (일렉트릭 블루) - '스톱', '대기' 상태
- **승리/획득:** `#00E676` (네온 그린) - 점수 획득, 미션 성공
- **경고/패널티:** `#FFB300` (앰버) - '뻑', '피박' 경고

### 2.3 텍스트 컬러
- **강조 (High):** `#FFFFFF` (흰색 100%)
- **본문 (Medium):** `#E0E0E0` (흰색 88%)
- **비활성 (Low):** `#9E9E9E` (회색 60%)

## 3. 타이포그래피 (Typography)
구글 폰트(Google Fonts)를 사용한다.

- **제목용 (Titles):** 'Do Hyeon' (도현체) - 굵고 힘이 있어 점수나 '고/스톱' 문구에 적합.
- **본문용 (Body):** 'Pretendard' (프리텐다드) - 가독성이 뛰어나고 현대적인 고딕체.
- **숫자 (Score):** 'Chakra Petch' 또는 'Russo One' - 아케이드 게임 느낌이 나는 폰트.

---

## 4. UI 컴포넌트 스타일 (Components)

### 4.1 버튼 (Buttons)
- **모양:** 둥근 모서리 (Radius 16px).
- **스타일:** 입체감이 살짝 있는 그라데이션 버튼.
  - '고' 버튼 예시: Red Orange 그라데이션 + 하단에 진한 그림자(3D 느낌).
- **인터랙션:** 버튼을 누르면 크기가 95%로 줄어들었다가 튕겨져 나오는(Bounce) 효과 필수.

### 4.2 화투패 컨테이너 (Card Widget)
- **테두리:** 평소에는 얇은 흰색 투명 테두리(`Colors.white24`).
- **선택 효과:** 낼 패를 선택하면 `#E6FF00` (형광 노랑) 색상의 빛나는 테두리(Glow)가 생겨야 함.
- **그림자:** 바닥 패와 구분되도록 깊은 그림자(`BoxShadow`) 적용.

### 4.3 다이얼로그 (Dialogs)
- **스타일:** 배경이 흐릿하게 비치는 반투명 유리창. (`BackdropFilter` 사용)
- **모서리:** 둥글게 처리 (Radius 24px).
- **테두리:** 아주 얇은 흰색 선으로 유리 모서리 느낌 강조.

---

## 5. 레이아웃 (Layout & Responsive)
모바일과 PC 웹을 모두 지원해야 한다.

### 5.1 모바일 (세로 모드)
- **상단:** 상대방 정보 (프로필, 현재 점수, 획득한 패 요약).
- **중앙:** 게임 판 (8장의 패가 깔림). 좁은 화면을 고려하여 그리드 형태로 배치.
- **하단:** 내 정보 (내 손패 10장을 부채꼴 혹은 겹쳐서 배치).

### 5.2 PC 웹 (가로 모드)
- **좌/우:** 플레이어 정보 패널을 양옆으로 배치 (넓은 공간 활용).
- **중앙:** 게임 판을 넓게 사용. 패의 크기를 키워서 시원하게 보여줌.

---

## 6. 애니메이션 및 효과 (FX)
- **패 나누기:** 중앙 덱에서 플레이어에게 0.1초 간격으로 착착착 날아가는 애니메이션.
- **쪽/뻑 이펙트:** 텍스트가 화면 중앙에서 '쾅' 하고 커졌다가 흔들리는(Elastic) 효과.
- **Lottie 활용:** 'GO', 'STOP' 결정 시 화려한 풀스크린 애니메이션 재생.
