## 1. 게임 로비 메뉴 안내

### 내 코인
화면 상단에 현재 보유 코인이 표시됩니다. 코인은 게임 참가, 아이템 구매 등에 사용됩니다.

### 광끼 게이지
코인 카드 아래에 광끼 게이지가 표시됩니다. 게이지가 100에 도달하면 특별한 광끼 모드를 활성화할 수 있습니다.

### 내 아이템
내가 구매한 아이템 리스트를 확인할 수 있습니다. 각 아이템은 게임 플레이 시 1회 사용 가능합니다.

### 일일 활동 메뉴
- **출석 체크**: 매일 출석하여 보상을 받을 수 있습니다
- **룰렛**: 하루 3회 기본 제공, 보너스 횟수 추가 가능
- **기부**: 코인을 기부하여 특별 보상을 받을 수 있습니다
- **랭킹**: 전체 플레이어 순위를 확인할 수 있습니다

### 아이템 샵
게임에서 사용할 수 있는 다양한 아이템을 구매할 수 있습니다. (판매 아이템은 매일 리셋됩니다.)

## 2. 1:1 멀티플레이 시작 방법

### 방 만들기
1. 로비 화면 하단의 **"방 만들기"** 버튼을 터치합니다
2. **"방 생성"** 버튼을 터치합니다
4. 상대방이 입장할 때까지 대기합니다

### 방 참가하기
1. 로비 화면에 표시되는 게임방 목록에서 참가할 방을 선택합니다
2. **"참가"** 버튼을 터치하여 게임에 입장합니다

## 3. 기본 룰 안내

### 게임 진행
1. 각 플레이어에게 10장의 패가 배분됩니다
2. 바닥에 8장의 패가 깔립니다
3. 자신의 턴에 손패 1장을 바닥에 내려놓습니다
4. 같은 월(花)의 패가 있으면 가져옵니다
5. 더미에서 1장을 뒤집어 같은 월의 패가 있으면 가져옵니다
6. 7점 이상 모으면 "고" 또는 "스톱"을 선택할 수 있습니다

### 패의 종류
- **광(光)**: 1월, 3월, 8월, 11월, 12월의 특별한 패 (총 5장)
- **열끗(動物)**: 각 월의 동물 그림 패 (총 9장)
- **띠(短冊)**: 각 월의 띠 그림 패 (총 10장)
- **피(カス)**: 일반 패 (총 24장)
- **보너스**: 쌍피 패 (총 2장)

### 고(Go)와 스톱(Stop)
- **고**: 게임을 계속하여 더 높은 점수를 노립니다 (점수 2배 증가)
- **스톱**: 현재 점수로 게임을 종료합니다

## 4. 특수 룰 안내

### 쪽/뻑/설사/따닥
- **쪽**: 내가 낸 패와 뒤집은 더미 패가 같은 월일 때, (상대편 피 1장 뺏어옴)
- **뻑**: 내 패와 바닥 패를 맞춘 후 뒤집은 패가 같은 월일 때 (바닥에 같은 월 3장이 모두 쌓임)
- **설사**: 바닥에 같은 월 패가 3장 있을 때, 모두 가져오기 (상대편 피 1장 뺏어옴)
- **따닥**: 바닥에 같은 월 패가 2장 있을 떄, 내가 낸 손패와 뒤집은 더미패로 4장을 모두 가져온 경우 (상대편 피 1장 뺏어옴)

### 폭탄
손패에 같은 월 패가 3장 있고 바닥에 같은 월 패가 1장 있는 경우. 모든 패를 가져갈 수 있습니다. (내 점수 x2, 상대편 피 1장 뺏어옴)

### 흔들기
손패에 같은 월 패가 3장 있으면 흔들어서 점수를 2배로 만들 수 있습니다.

### 총통
처음 받은 패 중 같은 월이 4장이 있으면 즉시 승리

## 5. 점수 계산 및 코인 정산

### 광 점수
| 조합 | 점수 | 설명 |
|------|------|------|
| 삼광 | 3점 | 비광(12월) 제외 광 3장 |
| 비광삼광 | 2점 | 비광 포함 광 3장 |
| 사광 | 4점 | 비광 제외 광 4장 |
| 비광사광 | 4점 | 비광 포함 광 4장 |
| 오광 | 15점 | 광 5장 모두 |

### 특수 조합 점수
| 조합 | 점수 | 설명 |
|------|------|------|
| 고도리 | 5점 | 2월, 4월, 8월 열끗 3장 |
| 홍단 | 3점 | 1월, 2월, 3월 홍단 3장 |
| 청단 | 3점 | 6월, 9월, 10월 청단 3장 |
| 초단 | 3점 | 4월, 5월, 7월 초단 3장 |

### 기본 점수
| 종류 | 기준 | 점수 |
|------|------|------|
| 열끗 | 5장부터 | 1점씩 추가 |
| 띠 | 5장부터 | 1점씩 추가 |
| 피 | 10장부터 | 1점씩 추가 |

### 배수 규칙
| 조건 | 배수 |
|------|------|
| 고(Go) 1회 | 2배 |
| 고(Go) 2회 | 4배 |
| 고(Go) 3회+ | +1배씩 추가 |
| 피박 | 2배 (상대 피 5장 이하) |
| 광박 | 2배 (상대 광 0장) |
| 멍따리기 | 2배 (상대 열끗 0장) |
| 고박 | 2배 (상대가 고 후 역전패) |

### 코인 정산
- 최종 점수 × 배수 × 베팅 금액 = 획득 코인
- 패배 시 베팅 금액만큼 코인 차감

## 6. 광끼 게이지 시스템

### 광끼 게이지란?
광끼 게이지는 게임 중 특정 행동을 통해 채워지는 특별한 게이지입니다.

### 게이지 충전 방법
게임 진행 시 쌓이게 되며 큰 점수차로 패배하게 된다면 더 많이 쌓이게 됩니다.

### 광끼 모드 발동
- 게이지가 100에 도달하면 **"광끼 모드"** 활성화 가능
- 광끼 모드 발동 시: **승자가 모든 베팅 코인을 가져감** (Winner Takes All)
- 광끼 모드 발동 시: 모든 플레이어는 Go 만 가능 (Stop불가)

### 주의사항
- 광끼 모드는 고위험 고수익 시스템입니다
- 발동 전 충분히 생각하고 결정하세요

## 7. 아이템 시스템

### 아이템 사용 규칙
- 아이템은 게임 당 1회, 자신의 턴에만 사용 가능합니다
- 모든 아이템은 최대 1개 까지만 구매 및 소유가 가능합니다

### 아이템 구매 방법
1. 로비에서 **"아이템 샵"** 버튼을 터치합니다
2. 원하는 아이템을 선택합니다
3. 코인을 사용하여 구매합니다
4. 구매한 아이템은 인벤토리에 저장됩니다
