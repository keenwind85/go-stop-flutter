## 사운드 파일 추가 방법

### 1. 파일 준비
- 형식: MP3 권장 (웹 호환성)
- 샘플레이트: 44100Hz 권장
- 비트레이트: 128kbps 이상 권장
- 파일명: 위 표의 파일명과 정확히 일치해야 함

### 2. 파일 복사
```bash
# 사운드 파일을 assets/sounds/ 폴더에 복사
cp your_sound_files/*.mp3 assets/sounds/
```

### 3. 사운드 활성화
`lib/services/sound_service.dart` 파일에서:
```dart
// 변경 전 (기본 음소거)
bool _isMuted = true;

// 변경 후 (사운드 활성화)
bool _isMuted = false;
```

### 4. 앱 재빌드
```bash
flutter clean
flutter pub get
flutter run -d chrome
```

## 볼륨 조절

게임 내에서 볼륨 조절이 가능합니다:
- 사운드 토글 버튼으로 음소거/음소거 해제
- 기본 볼륨: 70%
- BGM은 효과음의 50% 볼륨으로 재생

## 문제 해결

### 사운드가 재생되지 않는 경우
1. 파일명이 정확한지 확인 (대소문자 구분)
2. 파일이 `assets/sounds/` 폴더에 있는지 확인
3. `_isMuted = false`로 설정되어 있는지 확인
4. 브라우저에서 사운드가 차단되지 않았는지 확인

### 웹에서 사운드 지연이 있는 경우
- MP3 파일 크기를 줄이기 (128kbps 권장)
- 파일 길이를 짧게 유지
- 첫 사용자 인터랙션 후 사운드가 활성화됨 (브라우저 정책)
